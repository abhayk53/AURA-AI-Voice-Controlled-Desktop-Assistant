# 🤖 AURA AI - Voice Assistant

A desktop voice assistant built using Python that can perform tasks like opening applications, searching the web, and responding to voice commands with speech output.

## 🚀 Features
- Voice recognition using microphone  
- Text-to-speech response system  
- Opens applications (Chrome, Notepad, Calculator, VS Code, Steam)  
- Web navigation (Google, YouTube)  
- Real-time time updates  
- Interactive GUI with animated wave effect  
- Simple and user-friendly interface  

## 🛠️ Tech Stack
- Python  
- Tkinter (GUI)  
- SpeechRecognition  
- pyttsx3  
- PIL (Image Processing)  

## ▶️ Usage
1. Install required libraries:
```bash
pip install speechrecognition pyttsx3 pillow
```
🎯 Example Commands
"Open Chrome"
"Open YouTube"
"Open Notepad"
"What is the time"
"Open VS Code"
